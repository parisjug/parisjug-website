Paris JUG Academy le 13/1/2026

Les candidatures sont ouvertes !

La 12ème Paris JUG Academy aura lieu le 13/1/2026. Six personnes présenteront des sujets de 15mn, le public désignera alors un gagnant ou une gagnante. Cette session sera parrainée Audrey Neveu, Java Champion .
Les candidatures sont ouvertes jusqu’au dimanche 23/11/2025 (minuit à Paris).

  • - Tous les sujets sont acceptés ;
  • - La première candidature soumise est systématiquement acceptée ;
  • - Une candidature est tirée au hasard ;
  • - Nous annoncerons le résultat le 26/11/2025.

Les candidats peuvent bénéficier d'un accompagnement dans leur préparation lors de 4 soirées entre le 26/11/2025 et le 13/01/2026.
